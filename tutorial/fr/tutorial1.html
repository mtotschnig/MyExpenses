Gestion des comptes

En premier lieu, vous voudrez certainement modifier le compte par défaut ou en créer un nouveau. Activez l'entrée Sélectionner un compte du menu.

Comme toute application Android, un clic long sur le compte vous ouvre un menu contextuel, avec lequel vous pourrez éditer ou supprimer celui-ci. Le compte actuellement sélectionné ne peut pas être supprimé.

Entrez un libellé, une description, le solde d'ouverture et la devise de ce compte. La devise doit être saisie selon le code ISO 4217 en trois lettres, mais peut être facilement sélectionné à partir d'une liste en cliquant sur Sélectionner

Confirmez et vous reviendrez à la liste des comptes. Vous pourrez entrer autant de comptes que vous le désirez avec le menu Ajouter un compte. Pour sélectionner un compte, il suffit d'un clic court sur celui-ci dans la liste.
