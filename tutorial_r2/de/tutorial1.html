Konten

Beim ersten Aufruf von Meine Ausgaben wird ein Standardkonto angelegt, und Sie können sofort mit der Eingabe von Buchungen beginnen. Falls Sie das Standardkonto verändern wollen, verwenden Sie Einstellungen → Aktuelles Konto. Mittels Konten → Neu … hingegen können Sie ein neues Konto anlegen.

Geben Sie eine Bezeichnung, eine Beschreibung, den Anfangssaldo und die Währung des Kontos ein. Die Währung muss anhand ihrer dreistelligen Abkürzung ISO 4217 eingegeben werden, aber kann durch Klick auf Auswählen einfach aus einer Liste ausgewählt werden.

Nach Bestätigung der Eingabe, gelangen Sie wieder zur Liste der Buchungen. Wenn Sie ein neues Konto angelegt haben, wird dieses automatisch zum aktiven Konto. Der Schaltknopf Konten erlaubt das Wechseln zwischen Konton. Ein einfacher Klick wechselt zum zuletzt aktiven Konto, ein langer Klick öffnet ein Menü mit allen Konten zur Auswahl.

Die Verwaltung der Konten wird in Konten verwalten erklärt.
